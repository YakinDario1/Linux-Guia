# Lectura de archivo

COMANDO 

Comando `cat `

Muestra un archivo sin paginar. 

Comando `less` 

Muestra un archivo paginado . 

Comando `tail`

Muestra las ultimas 10 lineas de archivo. 

Comando `head`. 

Nuesrta las 10 primaras lineas de archivo. 

Comando `more `
